# Portfolio

Personal website and portfolio of **Gabe Wild**, a software developer with interests in full-stack development, systems, and data-driven applications.

This site serves as a central hub to showcase my projects, experience, and technical skills, as well as provide a way to learn more about my work and get in touch.

## Tech Stack

- **Framework:** Next.js  
- **UI Library:** Chakra UI  
- **Hosting:** Vercel  

## Features

- Responsive, modern design
- Project and experience highlights
- Optimized for performance and accessibility
- Deployed with continuous integration on Vercel

## Deployment

The site is automatically built and deployed via Vercel on every push to the main branch.
